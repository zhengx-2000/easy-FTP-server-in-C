## Implemented Function:
命令: Auth, PWD, CWD, LISt, MKDIR, DEL, RN <br>
Active mode 下载与上传 <br>
Display: Username, IP, Action, speed, total trafic. <br>


## How to use in Linux terminal
gcc myserver.c -o myserver -lcrypt  <br>
sudo ./myserver  <br>
